class quantarhei.qm.liouvillespace.systembathinteraction.SystemBathInteraction(sys_operators, bath_correlation_matrix, aggregate=None)[source]¶

Describes interaction of an open quantum system with its environment

Stores the system–bath interaction operator in form of a set of operators on the Hilbert space of the system and correlation functions of the operator on the bath Hilbert space.

Parameters:sys_operators (list) – List of the system part of the system-bath interaction Hamiltonian components
